Sentiment Analysis: LSTM vs BERT

Project Overview

A head-to-head comparison of a classical LSTM trained from scratch and a fine-tuned BERT transformer on binary sentiment classification, using 50,000 IMDB movie reviews.

Key Results

  • BERT: 92.64% accuracy, 92.64% F1 score, trained for 3 epochs
  • LSTM: 56.91% accuracy, trained for 10 epochs from scratch
  • Debugging: Diagnosed and resolved a vanishing gradient issue that initially stalled the LSTM at ~50% accuracy

Technologies Used

Python PyTorch HuggingFace Transformers scikit-learn

Impact

Demonstrated the value of transfer learning for NLP tasks — BERT's pretraining gave it a decisive advantage over a from-scratch model even with fewer training epochs.

Machine Translation: RNN vs MarianMT

Project Overview

Compared a GRU-based Seq2Seq model trained from scratch against a fine-tuned MarianMT transformer for English-to-French translation, using 50,000 sentence pairs from WMT14.

Key Results

  • MarianMT: 24.19 BLEU score after just 3 fine-tuning epochs
  • RNN Seq2Seq: 5.24 BLEU score after 10 epochs trained from scratch
  • Vocabulary tuning: Doubled vocabulary size from 10k to 20k words per language after diagnosing a high unknown-word rate in French

Technologies Used

Python PyTorch HuggingFace Transformers sacreBLEU
